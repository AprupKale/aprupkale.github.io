# Personal Webpage

This repository contains the source code for my personal academic webpage. The site serves as a central hub for my research, publications, projects, and professional background.

The webpage is intended for academic collaborators, recruiters, and others interested in my work in theoretical computer science, algorithmic game theory, fair division, and machine learning.

## Live Website

**URL:** https://aprupkale.github.io  

## Overview

The website includes:
- **About**: Academic background and research interests
- **Publications**: Selected papers and preprints, with links to full lists and external sources
- **Research**: Research projects that I have worked on (and am working on)
- **Contact**: Links to GitHub, LinkedIn, and email
